This page lists assorted educational materials I developed over the years. They range from simple sheet cheats to in-depth tutorials on structural bioinformatics. Feedback on existing content and suggestions for other tutorials are welcome.

* **Intermediate Python** ([Link](https://stanfordlibraries-carpentries.github.io/intermediate-python/)) ([Source](https://github.com/JoaoRodrigues/sc-teaching/tree/master/intermediate-python-2020))

    A 3-hour Python workshop that aims to familiarize participants with conventions and features of the language that are not necessarily covered in introductory courses, but that are nonetheless common in daily Python programming, by building a road trip planner.

* **Intro to Conda** ([Link](https://joaorodrigues.github.io/intro-to-conda/)) ([Source](https://github.com/JoaoRodrigues/intro-to-conda))

    A 3-hour Python workshop that aims to familiarize participants with conventions and features of the language that are not necessarily covered in introductory courses, but that are nonetheless common in daily Python programming, by building a road trip planner.

* **Git Cheat Sheet** ([Link](https://joaorodrigues.github.io/git-cheat-sheet/)) ([Source](https://github.com/JoaoRodrigues/git-cheat-sheet))

    A simple Git cheat sheet for personal use or for use in Git workshops. Can be rendered at HTML or printed as PDF.

* **Molecular Modeling and Simulation** ([Link](https://www.bonvinlab.org/education/molmod/)) ([Source](https://github.com/haddocking/molmod)) ([Data](https://github.com/haddocking/molmod-data))

    A syllabus developed for a MSc-level course on homology modeling, molecular dynamics simulation, and protein-protein docking. Originally developed by myself, partially based on existing tutorials by Tsjerk Wassenaar and Adrien Melquiond, these materials are now maintained by the [Bonvin lab](http://bonvinlab.org).
